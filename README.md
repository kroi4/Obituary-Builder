## 🌐 About (English)

A Hebrew-language obituary notice generator. This tool is designed specifically for creating traditional Jewish/Israeli obituary notices (מודעות אבל) in Hebrew.

### Features

- Full Hebrew RTL support
- Hebrew/Gregorian date conversion
- Multiple frame styles
- Gender-aware text adjustment
- Print and image export
- Mobile responsive design

### Usage

Simply open `index.html` in a browser and fill in the form.
